### Installation

Si vous voulez reprendre notre projet et le continuer.

Faites un git clone à partir de de l'adresse.

```text
git clone https://github.com/DavidNiembro/GoThroughFiles.git
yarn install
yarn start
```

`yarn start` pour lancer notre application.
